# Worldbuild Gen

课程 Sci-fi world 构建的辅助发散构思的世界观构建工具。

## 功能

- 10 个世界观分类：ENVIRONMENT, POLITICS, SOCIAL ACTORS, TECHNOLOGY, RELIGION, HISTORY, PHILOSOPHY, REFERENCE, X VARIABLE, ECONOMICS
- 每个分类下有多个子选项，支持随机生成
- 垂直布局，选项在每行内显示
- 极简黑白几何风格 UI
- 流体 blob 背景动画
- 支持连接 LLM API 生成叙事内容

## 使用

直接在浏览器中打开 `worldbuild.html` 即可使用。
